# DerecePanel

YKS Koçluk ve Öğrenci Yönetim Paneli. Next.js 16 (App Router), Tailwind CSS v4, Shadcn UI, Tremor.

## Kurulum

```bash
npm install
npm run dev
```

## Yapı

- `app/` — Next.js App Router sayfaları ve API route'ları
- `components/` — React bileşenleri (feature bazlı)
- `lib/` — İş mantığı, hook'lar, yardımcı fonksiyonlar
- `data/` — Kanonik veri kaynakları (YKS müfredat, YÖK Atlas)
- `styles/` — Test Maker CSS dosyaları
